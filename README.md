# 🎮 Top Earning Teams and Players in Esports | Tableau Dashboard

## 📌 Project Overview

The **Top Earning Teams and Players in Esports Dashboard** is an interactive Tableau dashboard that analyzes esports tournament earnings, team performance, player distribution, and game popularity. The dashboard enables users to explore prize money, team participation, and player statistics across different esports titles through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze total tournament earnings across esports games.
- Compare the number of teams participating in different games.
- Identify top-earning games and countries.
- Visualize player distribution across games.
- Provide interactive filtering for better exploration.
- Support esports business and market analysis.

---

## 🛠️ Tools & Technologies

- Tableau Desktop
- Microsoft Excel / CSV
- Tableau Calculated Fields
- Dashboard Actions
- Interactive Filters

The dataset contains historical esports tournament information, including:

## 📈 Dashboard Visualizations

### 1️⃣ Total Team Earnings by Game
- Displays total prize money earned by teams across different esports titles.
- Helps identify the highest revenue-generating games.

### 2️⃣ Total Teams by Game
- Compares the number of professional teams participating in each game.

### 3️⃣ Team Earnings by Genre
- Shows earnings grouped by game genre such as:
  - Battle Royale
  - MOBA
  - FPS
  - Card Games

### 4️⃣ Game Earnings Distribution
- Donut chart illustrating the percentage contribution of each game to the total prize pool.

### 5️⃣ Player Distribution Across Countries
- Scatter plot displaying player counts by country.

### 6️⃣ Player Earnings by Country
- Area chart comparing total prize earnings across countries.

### 7️⃣ Team Ranking by Number of Players
- Ranks teams based on total player count.

---

## 🎛 Interactive Filters

The dashboard includes the following filters:

- Genre
- Prize Money Range
- Game

These filters allow users to dynamically explore esports performance across multiple dimensions.

---

## 📈 Key Insights

- Dota 2 contributes the highest total tournament earnings.
- Battle Royale and MOBA genres dominate esports prize pools.
- Certain countries produce significantly more professional players.
- Team participation varies greatly across different games.
- A small number of games account for the majority of esports earnings.
